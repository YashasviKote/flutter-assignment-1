# Flutter Assignment 1 - Flutter Basics

## 📌 Objective

The objective of this assignment is to understand the Flutter project structure, learn Dart fundamentals, and build user interfaces using basic Flutter widgets.

---

## 📋 Tasks Completed

### ✅ Task 1 - Hello Flutter App
Created a simple Flutter application that displays:
- Hello Flutter
- Student Name
- Course

### ✅ Task 2 - Profile Card App
Developed a Profile Card using Flutter widgets:
- Container
- Column
- Image
- Text
- CircleAvatar

### ✅ Task 3 - Personal Info App (Mini Project)
Created a Personal Information application displaying:
- Profile Photo
- Name
- Course
- Skills
- Contact Information

## 🚀 How to Run

1. Clone the repository.

```
git clone <repository-url>
```

2. Open the project in Android Studio.

3. Install dependencies.

```
flutter pub get
```

4. Run the application.

```
flutter run
```
